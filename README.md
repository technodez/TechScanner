## 💻 درباره پروژه (About The Project)

در شبکه‌های محدود شده و پشت فیلترینگ (مانند شرایط کنونی اینترنت در ایران)، پیدا کردن آی‌پی‌های تمیز و سالم کلاودفلر یا دیگر CDNها برای اتصال به ابزارهای دور زدن فیلترینگ، یک چالش بزرگ است. 
بسیاری از اسکنرهای ساده به دلیل پدیده **Fake Handshake (پروکسی شفاف سیستم فیلترینگ)** فریب می‌خورند و پینگ‌های اعدادی مثل `3ms` را نشان می‌دهند که کاملاً فیک است.

پروژه **TechScanner** با یک مکانیزم پیشرفته و دو مرحله‌ای این مشکل را حل کرده است. این برنامه ابتدا باز بودن پورت را چک کرده و سپس با ارسال یک درخواست دیتای واقعی لایه TLS/HTTP، زمان دقیق رفت و برگشت پکت یا همان تاخیر واقعی (**Real RTT / TTFB**) را محاسبه می‌کند.

---

## ✨ قابلیت‌های کلیدی (Key Features)

- **Anti-Fake Ping (موتور ضد پینگِ فیک):** فیلتر کردن ۱۰۰٪ آی‌پی‌هایی که توسط ISP به صورت فیک باز نگه داشته شده‌اند.
- **Auto-ISP Detection:** تشخیص خودکار اینکه اینترنت شما همراه اول، ایرانسل، مخابرات، مبین‌نت یا شاتل است و بارگذاری رنج‌های اختصاصی همان اپراتور.
- **Manual Mode Loop:** امکان اجرای بی‌نهایت اسکن مجدد بدون نیاز به باز و بسته کردن برنامه، به همراه منوی دستی شیک.
- **Zero-Dependency (.EXE):** نسخه کامپایل شده بدون نیاز به نصب پایتون یا کتابخانه‌های جانبی روی ویندوز کار می‌کند.
- **Top 20 Sorting:** مرتب‌سازی نهایی نتایج در یک جدول بسیار تمیز بر اساس بهترین و پایدارترین پینگ واقعی.

## 🚀 راهنمای استفاده (How to Use)

### روش اول: استفاده از فایل آماده (پیشنهادی برای کاربران)
1. به بخش **[Releases](https://github.com/YOUR_USERNAME/TechScanner/releases)** در سمت راست همین صفحه بروید.
2. فایل `TechScanner.exe` را دانلود کنید.
3. آن را اجرا کرده و از اسکن هوشمند شبکه خود لذت ببرید!
